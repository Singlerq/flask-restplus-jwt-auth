# Flask-RestPlus 
## 简介:
0. 第三方模块flask-restplus(swagger doc)
1. 基于Flask Token 认证
2. 用户权限管理，API接口权限管理
## 主要组成:
0. Python3.6
1. Flask
2. Flask-RESTPlus
3. Flask-SQlAlchemy
4. ... 其他组件
## 运行
```
pip install -r ./requirement.txt
python manager
```

